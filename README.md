# Universiade Ticket

為了讓世大運訂票更方便而做的，台灣加油！

## Developement

```bash
# install package
npm install

# start
npm start

# test
npm test

# test in watch mode
npm run test:w
```

## Requirements

* Redis
* Nodejs v8.4
